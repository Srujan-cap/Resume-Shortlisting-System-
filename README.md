# Resume-Shortlisting-System-
Here, I developed a Resume Shortlisting System using Flask and Python to automate candidate screening based on required skills and job criteria. Implemented resume text extraction, keyword matching, and candidate ranking techniques to analyze resumes efficiently, while improving teamwork, leadership, and communication during project development.
Upload resumes in PDF and TXT formats
Extract resume text automatically
Skill-based keyword matching
Candidate score calculation
Automatic classification:
🔥 Priority Candidate
✅ Shortlisted
❌ Not Selected
Displays matched and missing skills
Stores shortlisted candidate details during runtime
Simple and user-friendly interface
